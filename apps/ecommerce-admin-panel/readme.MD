# Ecommerce Painel Administrativo

## Tecnologias
- Angular 18
- Angular Material
- Nx
- Jest

## Features
- Zoneless
- Signals
- Observables
- Variável de templates
- Angular Templates
- Angular Container
- Interceptors
- Guards

## Módulos
Módulo de administrador, com uma rota principal para a gestão de pessoas cadastradas da aplicação.
